About This Project

About Bangladesh Public Opinion Poll

This platform is a demonstration of web development technologies and is designed for educational purposes only. It simulates a public opinion poll for all 300 parliamentary constituencies of Bangladesh.

The system includes a complete voting interface, real-time results visualization, and a secure admin panel for managing candidates across all constituencies.

Technologies Used:

HTML5 Tailwind CSS JavaScript Firebase Firestore Chart.js

Important Disclaimer

This platform represents public opinion only and does not reflect official results of the Bangladesh Election Commission.

This website is a demonstration of web development technologies and is designed for educational purposes only.

Please Note:

  • This is NOT an official election website
  • Results shown here are NOT authentic election results
  • This platform does not collect personal identification information
  • Votes are limited to one per device for demonstration purposes only
  • All data is stored in a Firebase database and can be reset by administrators
  • Admin panel is secured with browser-based password storage
